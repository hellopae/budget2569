# บัญชีรายรับ-รายจ่าย 2569

โปรแกรมบันทึกรายรับ-รายจ่าย ปี พ.ศ. 2569

## Features
- ปฎิทินแบ่งรายเดือน 12 เดือน
- บันทึกรายรับ/รายจ่ายรายวัน พร้อมชื่อบริษัท/ลูกค้า
- สรุปยอดประจำเดือน (เงินเดือน, ค่าอาหาร, ค่าน้ำมัน, Easypass)
- พิมพ์/PDF ได้
- บันทึกข้อมูลใน LocalStorage (ไม่หายเมื่อปิดหน้าต่าง)
